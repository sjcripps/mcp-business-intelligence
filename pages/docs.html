Quick Start

Get up and running in under a minute:

  1. Create a free API key (no credit card required)
  2. Add the server config to your MCP client
  3. Ask your AI assistant to analyze a competitor, score a website, or research a market

Connection Setup

Add this configuration to your MCP client. Works with Claude Desktop, Cursor, Windsurf, VS Code with Copilot, and any MCP-compatible client.

Claude Desktop / Cursor / Windsurf

Add to your MCP settings file:

{
  "mcpServers": {
    "ezbiz-business-intelligence": {
      "url": "https://mcp.ezbizservices.com/mcp",
      "headers": {
        "X-API-Key": "your-api-key-here"
      }
    }
  }
}

Transport

The server uses Streamable HTTP transport (the modern MCP standard). Your client connects to https://mcp.ezbizservices.com/mcp via POST requests.

Authentication

Every request requires an API key passed as the X-API-Key header. Get your key from the signup page.

HeaderValue
X-API-KeyYour API key (starts with sk_biz_)

Tool: analyze_competitors

Analyzes the competitive landscape for a business. Returns competitor profiles, market positioning, SWOT analysis, and strategic recommendations.

Parameters

NameTypeRequiredDescription
business_namestringYesName of the business to analyze
industrystringYesIndustry or market sector (e.g., "SaaS", "local plumbing")
locationstringNoGeographic location for local businesses
website_urlstringNoThe business's own website URL for comparison

Example Prompts

  • "Analyze competitors for Acme Plumbing in Austin, TX"
  • "Who are the main competitors for Shopify in the e-commerce platform space?"
  • "Compare my SaaS startup (myapp.com) against competitors in project management"

Tool: score_web_presence

Scores a website's online presence from 0-100 across five categories: SEO, performance, content quality, social media presence, and trust signals. Returns detailed breakdown and prioritized recommendations.

Parameters

NameTypeRequiredDescription
urlstringYesWebsite URL to analyze
business_namestringNoBusiness name to check social media presence

Example Prompts

  • "Score the web presence of example.com"
  • "How strong is Apple's website from an SEO perspective?"
  • "Audit my website myshop.com and tell me what to fix"

Tool: analyze_reviews

Analyzes a business's online reviews and reputation. Aggregates sentiment, extracts themes, identifies strengths and weaknesses, and provides review management recommendations.

Parameters

NameTypeRequiredDescription
business_namestringYesBusiness name to search reviews for
locationstringNoBusiness location for local search
industrystringNoIndustry context for comparison

Example Prompts

  • "Analyze reviews for Joe's Pizza in Brooklyn, NY"
  • "What are customers saying about Zendesk?"
  • "Check the reputation of my dental practice, Bright Smiles Seattle"

Tool: market_research

Conducts market research on an industry. Returns market sizing (TAM/SAM/SOM), trends, opportunities, challenges, customer segments, and competitive landscape analysis.

Parameters

NameTypeRequiredDescription
industrystringYesIndustry or market to research
questionstringNoSpecific question to answer
locationstringNoGeographic focus for the research

Example Prompts

  • "Research the telemedicine market in the US"
  • "What's the TAM for residential solar in Texas?"
  • "Give me a market overview of the pet food industry"
